# 云平台子项目

## Environment Prepare

Install `node_modules`:

```bash
npm install
```
or
```bash
yarn
```

### 本地环境启动项目

```bash
yarn start
``` 
or 
```bash
npm run start
```

### 开发环境打包

```bash
npm run dev
```

### 测试环境打包

```bash
npm run test
```

### 生产环境打包

```bash
npm run prod
```


## 项目说明

### 技术栈
基于emp + react + antd4 + typescript

### 项目目录结构
- @types 全局类型、自定义方法
- config 
- src
  - components 公共组件
  - pages 页面
  - services 接口
  - utils 工具方法
- routes 路由列表
